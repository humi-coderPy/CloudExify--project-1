# 💰 Personal Expense Tracker

> **CloudExify Internship – Month 1 | Project 1**

## 📖 About the Project

The **Personal Expense Tracker** is a Python-based command-line application developed as part of the **CloudExify Internship Program (Month 1)**.

The application helps users manage their daily expenses by providing an easy-to-use menu-driven interface. Users can record expenses, organize them into categories, update or delete existing records, and view their spending history efficiently.

This project demonstrates the practical implementation of Python fundamentals, modular programming, and Object-Oriented Programming (OOP).
