# CloudExify--project-1
CloudExify internship - project 1



## Project Overview
This is a console-based Expense Tracker application developed in Python for CloudExify Internship Project 1. The application implements core Python concepts including functions, file handling, data validation, and persistent storage to help users track daily expenses. Git and GitHub were used for version control throughout the development lifecycle.

## What I Did
1. Configured Git and created a GitHub repository for project management
2. Developed a Python-based Expense Tracker with modular functions
3. Implemented file I/O operations for storing and retrieving expense data
4. Added features for expense categorization and total calculation
5. Managed source code using Git commits and pushed updates to GitHub

## Features
- Add new expense with amount, category, and date
- View complete expense history
- Calculate total expenditure
- Filter expenses by category
- Data persistence using text file storage

## Technologies Used
- Python 3.x
- File Handling & Exception Handling
- Git & GitHub
- VS Code

## How to Run
1. Ensure Python 3.x is installed
2. Clone this repository
3. Run the command: `python expense_tracker.py`

## Learning Outcome
This project strengthened my understanding of Python fundamentals, file handling mechanisms, and modular programming. Additionally, I gained practical experience with Git workflows, repository management, and collaborative development practices using GitHub.
